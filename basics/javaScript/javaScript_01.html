# P1-Converting and Checking Numbers

Numbers

在 JavaScript 中，所有的数字（整数和小数）存储在内存中的方式都是浮点数。由于小数进行转换和计算时采用的是二进制浮点数格式，因此可能会出现精度问题，因此0.1+0.2!=0.3。

  • Number.parseInt() 方法用来将给定的字符串参数转换成整数类型并返回这个整数。第一个参数为字符串，第二个参数为需要转换的进制（可选），如果字符串无法解析返回NaN。如果第一个数字字符可以被转换成数字，则 Number.parseInt() 方法会继续解析字符串中的其它字符，直到遇到无法被解析的字符为止。
console.log(23 === 23.0); // True

// Base 10 - 0 to 9. 1 / 10 = 0.1 10 / 3 = 3.333333
// Binary base 2 - 0 1
console.log(0.1 + 0.2); // 0.30000000000004
console.log(0.1 + 0.2 === 0.3); // false

// Conversion
console.log(Number("23")); // 23
console.log(+"23"); // 23

// Parsing
console.log(Number.parseInt("30px", 10)); // 30
console.log(Number.parseInt("e23", 10)); // NaN

console.log(Number.parseInt(" 2.5rem ")); // 2
console.log(Number.parseFloat(" 2.5rem ")); // 2.5
// console.log(parseInt(' 2.5rem '));

// Check if value is NaN
console.log(Number.isNaN(20));
console.log(Number.isNaN("20"));
console.log(Number.isNaN(+"20X"));
console.log(Number.isNaN(23 / 0));

// Checking if value is number
console.log(Number.isFinite(20));
console.log(Number.isFinite("20"));
console.log(Number.isFinite(+"20X"));
console.log(Number.isFinite(23 / 0));

console.log(Number.isInteger(23));
console.log(Number.isInteger(23.0));
console.log(Number.isInteger(23.3));
console.log(Number.isInteger(23 / 0));

# P2-Math and Rounding

Math

Math数学函数。

  • Math.max 会强制转换
console.log(Math.sqrt(25));
console.log(25 ** (1 / 2));
console.log(8 ** (1 / 3));

console.log(Math.max(1, 2, 3, 23, 1));
console.log(Math.max(2, 18, "23", 11, 2));
console.log(Math.max(5, 18, "23px", 11, 2));
console.log(Math.min(5, 2, 3, 13, 3));

console.log(Math.PI * Number.parseFloat("10px") ** 2);

Math.random

Math.random()产生[0,1)之间的数。

console.log(Math.trunc(Math.random() * 6) + 1);

[11, 20];
const randomInt = (min, max) =>
  Math.trunc(Math.random() * (max - min) + 1) + min;
// 0...1 -> 0...(max - min) -> min...max
console.log(randomInt(10, 20));

取整

  • Math.round()四舍五入
  • Math.ceil()向上取整
  • Math.floor()向下取整
  • Math.trunc()直接取整
// Rounding integers
console.log(Math.round(23.3));
console.log(Math.round(23.9));

console.log(Math.ceil(23.3));
console.log(Math.ceil(23.9));

console.log(Math.floor(23.3));
console.log(Math.floor("23.9"));

console.log(Math.trunc(23.3));

console.log(Math.trunc(-23.3));
console.log(Math.floor(-23.3));

保留指定位数小数

toFixed保留指定位数小数并返回一个字符串，会四舍五入。

// Rounding decimals
console.log((2.7).toFixed(0));
console.log((2.7).toFixed(3));
console.log((2.345).toFixed(2));
console.log(+(2.345).toFixed(2));

# P5-Working with BigInt

BigInt

ECMAScript2020推出的存储任意大数字。

  • 可以通过在大数字后加 n 表示 BigInt
  • 也可以通过BigInt(Number)转换成 BigInt，当 number 过大时会不精准
  • BigInt 只能和 BigInt 数学运算
console.log(2 ** 53 - 1);
console.log(Number.MAX_SAFE_INTEGER);
console.log(2 ** 53 + 3); // 精度不对

console.log(233223939393939393999993333323232n);
console.log(BigInt(233223939393939393999993333323232)); // BigInt转换大数字时候可能不精准，因为先处理的数字
//console.log(Math.sqrt(16n)); error

// Operations
console.log(1000n + 1000n);
console.log(233333324234242342342424n * 10000n);

const huge = 23333333332222222200023n;
const num = 23;
// console.log(huge * num); // error
console.log(huge * BigInt(num));

console.log(20n > 15); // true
console.log(20n === 20); // false
console.log(typeof 20n); // bigint
console.log(20n == "20"); // true

console.log(huge + " is REALLY big !!!");

// Divesions
console.log(11n / 3n); // 3n
console.log(10 / 3);

Add Date

const now = new Date();
const day = `${now.getDate()}`.padStart(2, 0);
const month = `${now.getMonth() + 1}`.padStart(2, 0);
const year = now.getFullYear();
const hour = now.getHours();
const min = now.getMinutes();
labelDate.textContent = `${day}/${month}/${year}, ${hour}:${min}`;
