# Markdown 日常写做

# Markdown 能干什么？

markdown 是一种轻量级标记语言，用于在文本文档中添加格式化元素。主要特点：语法简单、易上手、易学习。

简单来说markdown可以用来轻松的对文档进行排版，很多应用都支持该语法如：

  • CSDN、掘金、简书等博客平台
  • Notion、语雀、Typora、腾讯文档等笔记或知识库软件
  • 代码托管平台、开源项目、开发文档都用来写作

# 为什么要用 Markdown 来写作？

"欲善其事，必先利其器"，它非常的强大能够快速的排版文档、且支持自定义主题来定义自己的文档，同时 Markdown 非常适合技术文档。它为做笔记和为网站创建内容提供了便捷的语法。

  • .md 文档能够转 .html 网页 和 .pdf 文件
  • 像 Vuepress、Wordpress 等博客支持 .md 文件写作

文档创做工具推荐：

  • Vscode、Sublime Text、Typora
  • 在线 Markdown 工具open in new window
